# lung-cancer-gene-expression-analysis
Teknologi biologi molekuler memungkinkan peneliti mempelajari perubahan genetik pada sel kanker melalui analisis ekspresi gen. Metode ini digunakan untuk mengidentifikasi gen yang mengalami peningkatan (*upregulated*) atau penurunan (*downregulated*) ekspresi pada jaringan kanker dibandingkan jaringan normal.
